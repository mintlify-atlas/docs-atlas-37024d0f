MapLibre Native is a cross-platform C++ library that powers interactive vector maps in native applications. It renders maps using the GPU through multiple graphics backends — OpenGL ES, Metal, Vulkan, and WebGPU — giving you fast, smooth map experiences on Android, iOS, Linux, Windows, macOS, and Node.js.

## Why MapLibre Native

MapLibre Native originated as a community-led fork of Mapbox GL Native after Mapbox switched to a non-open-source license in December 2020. It is licensed under the BSD 2-Clause License and developed entirely in the open.
